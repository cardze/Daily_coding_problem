# First Recurring Character

**Date:** 2023-12-04  
**Asked by:** Google

## Problem Description

Given a string, return the first recurring character in it, or null if there is no recurring character.

## Examples

**Example 1:**
- Input: `"acbbac"`
- Output: `"b"`

**Example 2:**
- Input: `"abcdef"`
- Output: `null`
