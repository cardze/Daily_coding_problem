# One-to-One Character Mapping

**Date:** 2023-12-21

## Problem Description

Determine whether there exists a one-to-one character mapping from one string s1 to another s2.

## Examples

**Example 1:**
- Input: s1 = `"abc"`, s2 = `"bcd"`
- Output: `true`
- Explanation: We can map a→b, b→c, and c→d

**Example 2:**
- Input: s1 = `"foo"`, s2 = `"bar"`
- Output: `false`
- Explanation: The 'o' cannot map to two different characters
